# Version C: Void Nomads

This alternate continuation begins after the shared canon of Chapters 1-20 and carries the Lars Odin novel through Chapter 55. Version C emphasizes hard-sci-fi sense of wonder: the KBC Void as a low-observation quiet buffer, Creators as portal builders who misunderstood older systems, and Multiverse Nomads as field-like intelligences and engineers rather than gods.

Reading order: start with the shared opening Chapters 1-20, then read these files in numeric order from `21-containment-drill.md` through `55-a-quiet-signal.md`.
