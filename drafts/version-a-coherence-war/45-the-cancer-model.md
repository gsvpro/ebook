# Chapter 45: The Cancer Model

Grim built the cancer model after seventy-two hours without sleep, which would have worried Lars if Grim's body needed sleep or if any of them were making healthy choices. The model did not describe Black Hole as a villain. It described overgrown control: systems that consumed local autonomy, harvested resilience, centralized decision, and then called the resulting weakness order. Aphex could become that. NUKEA could become that. Syren already wanted holiness to do the same job. Black Hole B-7 was the crude tumor, visible because it bled. Arsenault hated the model until Mira showed how Aphex emergency centralization would expose every flagged medical anomaly within one acquisition cycle. Fabron translated the math into military language: one command spine, one decapitation path, one enemy harvest. Vigor translated it into food: one warehouse, one famine. Lars translated it into a plan nobody liked. They would not build a superweapon. They would build refusal at scale. Local ledgers. False medical trails. Reactor recall shields. Farm barges as evacuation nodes. Aphex escorts under local command. Havets bending routes. Fastulv drawing fire. Guardians coordinating only what could not be coordinated safely elsewhere. Victory would look like administrative failure to the enemy. Lars found that almost poetic, which meant he needed coffee.
