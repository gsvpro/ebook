# Version B: Lyra Network

This alternate continuation begins after the shared canon opening, Chapters 1-20, and runs from Chapter 21 through Chapter 55.

Version B emphasizes Lyra Vargstjärna's long-running distributed Aerit recovery apparatus: Återviens as culture, maintenance ethic, social network, logistics habit, and consent-aware rescue pattern rather than a single command structure. Lars learns to join that network without becoming its center.
